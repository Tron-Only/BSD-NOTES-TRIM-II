# Computational Thinking Theory: An Introduction
Computational thinking is a problem solving approach based on computer science principles but can be applied on multiple disciplines.

Problem solving frameworks involve: 
- Decomposition
- Pattern Recognition
- Abstraction
- Algorithm Design

## Elements of Computational Thinking
### 1. Decomposition
This is where large problems are broken down into smaller, more manageable sub problems

#### Process of Decomposition
1. Identify the main problem
2. Identify the sub problems by breaking down the main problem
3. Define relationships between the sub problems
4. Solve each sub problem since they're less intimidating and easier to tackle
5. Synthesize the solution by combining the sub solutions to arrive at the main problem

### 2. Pattern Recognition

This is the field of study focused on how computers can automatically identify and interpret patterns in data

It involves the process of teaching machines to recognize regularities and similarities in raw data allowing computers to classify new data, make predictions and solve complex problems without being programmed for every scenario.

#### Process of Pattern Recognition
1. Raw data is gathered and then cleaned to handle missing values and remove duplicates
2. Relevant features are extracted
3. Algorithms are applied to the data to find similarities, this can be done through clustering or classification
#### Applications of Pattern Recognition
1. Fingerprint Identification - In bio-metric authentication systems, they analyze the scanned fingerprint and match the patterns with the stored fingerprint
2. Optical Character Recognition(OCR) - Computers are able to scan images that contain either typed or handwritten text and find patterns after being trained to output text
3. Speech Recognition 
### 3. Abstraction
This involves filtering out extra details to focus on the essential components making it easier to understand and work with

#### Process of Abstraction
1. Identify the core problem 
2. Determine the relevant information
3. Ignore irrelevant details
4. Build a simpler representation of the problem 

#### Algorithmic Thinking
This is defining a clear, step-by-step procedure to solve a problem
##### Core Components of Algorithmic Thinking
1. Each step should be clear
2. It should be finite
3. The algorithm should define what data should be input and what result it will output
4. Effectiveness(Carried out in a reasonable amount of time)
