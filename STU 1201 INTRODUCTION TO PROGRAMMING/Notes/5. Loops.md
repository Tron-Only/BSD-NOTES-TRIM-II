# Introduction to Programming: Loops in C
In programming there are situations where the task needs to be completed multiple times.
Writing the same statement multiple times, would make the program long and inefficient.
To solve this problem, C provides loops(Iteration statements) that allow a block of code to be iterated repeatedly as long as a specified condition is true
A loop helps programmers automate repetitive tasks.

**Loop** - Is a control structure that repetitively executes a block of code while a certain  condition remains true.

## Why loops are important
1. Avoids repeating code many times
2. Makes programs shorter and easier to read


There are 3 types of loops in C:
1. For loop
2. While loop
3. Do-while loop


## For loop
Is used when the number of iterations is known beforehand

```C
for (initialization; condition; update){
	// Code Block
}
```
Initialization - Starting value of the loop
Condition - Determines whether the loop continues
Update -  changes the loop variable after each iteration(increment or decrement)

Write a C program using `for loop` to print numbers from 1 to 10 vertically

```C
#include <stdio.h>

int main() {
  for (int number = 1; number <= 10; number++) {
    printf("%d \n", number);
  }
  return 0;
}
```

Write a C program to print numbers from 20 to 100 horizontally

```C
#include <stdio.h>

int main() {
  printf("--- Program to print numbers horizontally (20-100---)");
  for (int number = 20; number <= 100; number++) {
    if (number == 100) {
      printf("100");
      break;
    }
    printf("%d, ", number);
  }
  printf("\n");
  return 0;
}
```

Write a program in C to print numbers from 50 to 30 horizontally

```C
#include <stdio.h>

int main() {
  printf("--- Program to print numbers horizontally (50-30) ---\n");
  for (int number = 50; number >= 30; number--) {
    printf("%d, ", number);
  }
  printf("\n");
  return 0;
}
```

Write a C program to print even numbers from 20 - 50
```C
#include <stdio.h>

int main() {
  for (int number = 20; number <= 50; number++) {
    if (number % 2 != 0) { //odd numbers 
      printf("%d\n", number);
    }
  }
  return 0;
}
```

Write a C program that prints the sum of numbers from 1 to 10

```C
#include <stdio.h>

int main() {
  int sum;
  for (int i = 1; i <= 10; i++) {
    sum = sum + i;
  }
  printf("The sum of the digits from 1 - 10: %d\n", sum);
  return 0;
}
```
