# Introduction to Programming: Switch
The switch statement in C is a decision making control structure used to execute one block of code from multiple options.
It is mainly used when he variable has many possible values and we want to perform different expressions depending on the value
The switch statement is often used as a statement to `if-else-if` statements especially when checking a single variable against many constant values

*NB: READ ABT CONSTANTS*


## Syntax of Switch-Statements
```C
switch(expression) {
	case value1:
		// Code Block
		break;
	case value2:
		// Code Block
		break;
	case value3:
		// Code Block
		break;
	default:
		// Code Block (If none of the earlier statements are true)
}
```

Example:

Write a C program, to print a simple menu program for the user to enter numbers between 1 and 3. (Use Switch)

```C
#include "stdio.h"
int main() {
int chosen_input;
printf("Enter a number of your choice (1-3): ");
scanf("%d", &chosen_input);
switch (chosen_input) {
	case 1:
		printf("You chose the 1st option!\n");
		break;
	case 2:
		printf("You chose the 2nd option!\n");
		break;
	case 3:
		printf("You chose the 3rd option!\n");
		break;
	default:
		printf("Invalid choice. %d is not a valid input \n", chosen_input);
}
}
```

Write a C program using switch statements to check the day of the week, the number entered by the user, represents the day of the week.

```C
#include "stdio.h"

int main() {
  int chosen_input;
  printf("Enter the number of the week (1-7): ");
  scanf("%d", &chosen_input);

  switch (chosen_input) {
  case 1:
    printf("Monday\n");
    break;
  case 2:
    printf("Tuesday\n");
    break;
  case 3:
    printf("Wednesday\n");
    break;
  case 4:
    printf("Thursday\n");
    break;
  case 5:
    printf("Friday\n");
    break;
  case 6:
    printf("Saturday\n");
    break;
  case 7:
    printf("Sunday\n");
    break;
  default:
    printf("Invalid choice. %d is not a day of the week \n", chosen_input);
  }
}

```
