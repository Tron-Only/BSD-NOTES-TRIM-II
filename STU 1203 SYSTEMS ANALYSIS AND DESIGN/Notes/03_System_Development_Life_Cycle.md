# SDLC: System Development Life Cycle
The System Development Life Cycle (SDLC) is a model for designing a system. 

It is required by dynamic business that need dynamic information systems to change with the requirements of the business. Therefore it requires an information system to not be static.

## Phases of SDLC
1. Planning Phase
2. Design Phase
3. Analysis Phase
4. Implementation Phase
5. Maintenance Phase
